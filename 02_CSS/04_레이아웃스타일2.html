요소 정렬 스타일(float / clear)

        float (뜨다, 띄우다): 요소를 띄워서 좌/우로 정렬하는 속성

        clear: float로 인해 띄워져 있는 상태를 해제하는 속성
                (float로 인해 겹쳐지는 현상을 제거)

                -> float : left;로 인한 겹침현상을 제거하기 위해서는 
                            clear : left; 를 작성해야한다.(방향성이 같아야 한다.)
