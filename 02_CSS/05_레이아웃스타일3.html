여백 관련 속성

        HTML 요소는 총 네 가지의 영역으로 구성되어 있다.

        1) content 영역 : 요소의 내용이 작성되어지는 영역
        
        2) border 영역 : 요소의 테두리가 지정되는 영역
        
        3) padding 영역 : content와 테두리 사이 영역

        4) margin 영역 : 다른 요소와의 간격을 나타내는 영역
