영역 분할을 위한 기본 구조 설정 방법

영역 관련 속성
1) content: 내용이 표시되는 영역
2) padding: content와 border 사이 영역
3) border: 요소의 테두리가 표시되는 영역
4) margin: 다른 요소와의 간격을 지정하는 영역
width : 300px; height : 200px 지정한 상태
-> content 영역의 크기

실제 현재 요소의 크기는
content + padding + border =
가로: 300 + (20*2) + (20*2) = 380px
세로: 200 + (20*2) + (20*2) = 280px
