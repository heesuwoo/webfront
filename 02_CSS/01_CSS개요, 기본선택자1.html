CSS란?

        Cascading Style Sheets 의 약자로

        마크업 언어가 실제로 화면에 표시되는 방법(모양, 색, 스타일)을 기술하는 언어.
        (html코드가 화면에 어떤 모양, 색으로 보여질지 지정하는 언어)

        W3C 표준으로 지정되어 있음.

CSS 기본 선택자

        선택자(selector)란?
        현재 HTML 문서 내에 특정 요소를 선택하는 문자
        
        선택된 요소에 원하는 스타일을 적용할 수 있다.

태그 선택자

        현재 HTML 문서 내에 같은 태그를 선택하는 선택자

        [작성법]
        span { css 코드 }
span 태그 내부에 작성된 글자입니다. strong 태그 내부에 작성된 글자입니다. span 태그 내부에 작성된 글자입니다.

ID 선택자 (#id속성값)

        HTML 문서 내에 id 속성값이 일치하는 요소를 선택하는 선택자.
        (중요) id 속성 값은 페이지 내에서 유일해야 한다! (중복 X)

        [작성법]
        #id 속성값 { css 코드 }

        (# == id)

class 선택자 (.class 속성값)

        HTML 문서 내에 같은 class 속성 값을 가지는 태그를 모두 선택하는 선택자.

        [작성법]
        .class 속성값 { css 코드 }
        (. == class)

모든 요소 선택자(*)

        단독 또는 다른 선택자와 함께 사용 가능한 선택자

        단독: 현재 HTML 문서 내 모든 요소 선택

        다른 선택자와 같이: 해당 선택자 내부, 형제, 부모 등 모두 선택
