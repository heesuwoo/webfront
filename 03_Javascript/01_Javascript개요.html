script 언어란?

        응용 프로그램의 동작을 사용자의 요구에 맞게 수행되도록 제어하는 용도의 언어.
        
        별도로 소스코드의 컴파일을 진행하지 않고
        인터프리터(Interpreter)를 이용하여 소스코드를 한 줄씩 읽어 바로 실행함.

        * 인터프리터(Interpreter)
            - 소스코드 컴파일을 하지 않고 바로 수행
            - 컴파일에 소요되는 시간을 없지만,프로그램 수행에 시간이 소요됨.

        ex) 브라우저(응용 프로그램)를 제어하는 자바스크립트

Javascript (JS) 란?

        웹 브라우저에서 많이 사용하는 인터프리터 방식의 객체 지향 프로그래밍 언어.
        ECMA Script 표준을 따르는 대표적인 웹 기술이다.
        (ECMA: European Computer Manufacturers Association)

        * JS는 우리가 알고있는 프로그래밍 언어의 기본 규칙이 적용되지 않는 경우가 많다.
        ex) 변수 자료형, 전역/지역(var, 클로저), HTML/CSS를 다루는 명령어
